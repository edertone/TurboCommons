To publish the library to npm:



1 - Generate a release build
2 - Create a folder on your desktop and add all files from dist/NNN/
3 - Copy the package.json file from this folder to the created desktop folder
4 - Update the version number on the package.json file
5 - Update the project name on the package.json file
6 - Open a command line inside the created desktop folder and run : npm publish
